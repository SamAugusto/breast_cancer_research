# Data information

Sources
Institution: University of Wisconsin Hospitals, Madison, Wisconsin, USA. Creators: Dr. William H. Wolberg (physician), W. Nick Street, and Olvi L. Mangasarian (computer scientists) at the University of Wisconsin, Madison. Donation: The dataset was donated to the UCI Machine Learning Repository on July 15, 1992, with periodic updates reflecting additional clinical cases. Availability:

UCI Machine Learning Repository: https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic Kaggle: https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data scikit-learn: Accessible via the load_breast_cancer() function in Python.

Licensing: Licensed under Creative Commons Attribution 4.0 International (CC BY 4.0), allowing sharing and adaptation with proper attribution. Related Dataset: The dataset is distinct from the Breast Cancer Wisconsin (Original) dataset, also from the University of Wisconsin, which focuses on recurrence prediction and contains 699 instances with 9 attributes.
